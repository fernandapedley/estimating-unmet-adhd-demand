# Estimating Potential Unmet ADHD Assessment Demand in England

## Overview

This capstone project explores ADHD prevalence, referrals and waiting-list activity in England using data published by NHS England.

The project investigates whether trends in prevalence, referrals and waiting-list activity can provide useful insight into potential unmet demand for ADHD assessment and treatment services.

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- GitHub Pages

## Data Source

NHS England Neurodevelopmental Data Hub
