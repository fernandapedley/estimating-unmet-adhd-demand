## Overview

Attention Deficit Hyperactivity Disorder (ADHD) services across England are experiencing increasing demand, with many patients facing lengthy waits for assessment and treatment.

This capstone project uses publicly available NHS England data to explore ADHD prevalence, referral activity and waiting-list trends. By comparing observed service activity with estimated population need, the analysis aims to provide insight into potential unmet demand for ADHD assessment services across England.

The project demonstrates how data analytics techniques can be applied to a real-world healthcare challenge using publicly available datasets.

---

## Project Objectives

- Estimate expected ADHD demand using published prevalence estimates.
- Analyse referral and waiting-list trends using NHS England data.
- Compare estimated need with observed service activity.
- Identify evidence of potential unmet demand.
- Demonstrate the application of data analytics techniques within a healthcare setting.

---

## Key Questions

This analysis explores the following questions:

1. What level of ADHD assessment demand might be expected based on published prevalence estimates?
2. How have referrals and waiting-list indicators changed over time?
3. To what extent does observed service activity reflect estimated population need?
4. What do these findings suggest about potential unmet demand for ADHD assessment services in England?

## Data Sources

The analysis draws on publicly available datasets, including:

- NHS England ADHD Management Information (MI-ADHD) dataset
- NHS England Neurodevelopmental Data Hub
- Office for National Statistics (ONS) population estimates
- Published ADHD prevalence research

Further information about data preparation, assumptions and limitations can be found in the Methodology section.
