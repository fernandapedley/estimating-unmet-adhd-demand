## Data Sources

### NHS England ADHD Management Information (MI-ADHD)

Monthly ADHD prevalence, referral and waiting-list activity data published by NHS England.

[NHS England ADHD Management Information (MI-ADHD)](https://digital.nhs.uk/data-and-information/publications/statistical/mi-adhd)

---

### NHS England Neurodevelopmental Data Hub

The primary dataset used in this project was obtained through the NHS England Neurodevelopmental Data Hub.

[NHS England Neurodevelopmental Data Hub](https://digital.nhs.uk/data-and-information/data-tools-and-services/data-services/neurodevelopmental-data-hub)

---

### ADHD Data Improvement Plan

Supporting documentation relating to improvements in ADHD data collection, reporting quality and data completeness.

[ADHD Data Improvement Plan](https://digital.nhs.uk/data-and-information/data-tools-and-services/data-services/neurodevelopmental-data-hub/adhd-data-improvement-plan)

## Project Outputs

The project produced:

* A fully documented analytical report
* A reproducible Jupyter Notebook
* A GitHub Pages portfolio website
* Data visualisations exploring ADHD prevalence, referrals and waiting-list activity
* A waiting-list-to-prevalence ratio as a proxy indicator of service pressure
* Recommendations relating to ADHD data quality, adult ADHD pathways, unmet demand measurement and future research

This project was completed for educational purposes as part of the Imperial College London Executive Professional Certificate in Data Analytics.
